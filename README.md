# Advanced Telegram Contact Bot 🤖

A powerful Telegram bot that collects detailed device information when users share their contacts.

## Features ✨
- 📱 Collect contact information
- 🔍 Advanced phone model detection
- 🔋 Battery status monitoring
- 🌐 Network details (IP, MAC, WiFi)
- 💾 Automatic backups
- 📊 Statistics tracking

## Commands 📝
- `/start` - Start the bot
- `/help` - Show help
- `/status` - Check bot statistics
